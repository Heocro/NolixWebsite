Nolix sequencer

Can run functions in background.
ch.nolix.core.sequencer

How to run a function in background

Sequencer.runInBackground(() -> doCalculations());

How to check if a background run is finished

var future = Sequencer.runInBackground(() -> doCalculations());
...
if (future.isFinishedSuccessfully()) {
  ...
}

How to wait for 500 milliseconds

Sequencer.waitForMilliseocnds(500);
