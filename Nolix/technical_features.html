Technical Features

Nolix provides a wide range of technical features for Java.
The technical features of Nolix feel comfortable - not technical.


Nolix containers Provides containers with various search methods.
Nolix Logger Provides a Logger that can log messages, values, warnings and errors.
Nolix Sequencer Provides a Sequencer that can run methods in background.
Nolix tests Provides base classes for unit tests and integration tests.
Nolix Validator Provides a Validator that can validate arguments.
