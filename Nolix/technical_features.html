Technical Features

The technical features of the Nolix library are features with a general purpose like validators or containers. The technical features can be useful for nearly all Java projects.


Nolix containers Provides containers with various search methods.
Nolix logger Can log messages, values, warnings and errors.
Nolix sequencer Can run functions in background.
Nolix tests Provides base classes for unit tests and integration tests.
Nolix validator Provides functions to validate arguments.
