Nolix console clients

Description

 Provides console applications as server-client-programs.

Packages

 ch.nolix.system.consoleClient

Tutorials

 ConsoleClientTutorial


Create a NetServer

var netServer = new NetServer(50000);

Create an Application for BackConsoleClients

var application = new Application<>("Demo", BackConsoleClient.class, MainSession.class);

netServer.addApplication(application);

Define a Session for BackConsoleClients

public class MainSession extends BackConsoleClientSession {
  public void initailize() {
    writeLineToConsole("Welcome!");
    writeLineToConsole("Enter your name.");
    var name = readLineFromConsole();
    writeLineToConsole("Hi " + name + " .");
  }
}

Create a FrontConsoleClient that connects from the same program

var frontConsoleClient = new FrontConsoleClient(server, "Demo");

Create a FrontConsoleClient that connects from the network

var frontConsoleClient = new FrontConsoleClient("127.0.0.1", 50000, "Demo");

Set the look of a BackConsoleClient

public class MainSession extends Session<BackConsoleClient> {

  }
}
