Nolix console clients

Description

 Provides console applications as client-server-programs.

Packages

 ch.nolix.system.consoleClient

Tutorials

 BackConsoleClientTutorial


How to create a Server

Server server = new Server(50000);

How to create an Application for BackConsoleClients

StandardApplication<BackConsoleClient> application = new StandardApplication<BackConsoleClient>("Demo", MySession.class);

server.addApplication(application);

How to define a Session for BackConsoleClients

public class MySession extends Session<BackConsoleClient> {
  public void initailize() {
    writeLineToConsole("Welcome!");
    writeLineToConsole("Enter your name.");
    var name = readLineFromConsole();
    writeLineToConsole("Hi " + name + " .");
  }
}

How to create a FrontConsoleClient

FrontConsoleClient frontConsoleClient = new FrontConsoleClient("200.200.200.200", 50000, "Demo");

How to set the look of a BackConsoleClient

public class MySession extends Session<BackConsoleClient> {
  public void initailize() {
    setLook(new BlackGreenFrontConsoleClientLook());
  }
}
