Nolix console clients

Provides console applications as client-server-programs.
ch.nolix.system.consoleClient
Console client tutorial

How to create a server

var server = new Server(50000);

How to create a console client application

var application = new StandardApplication<ConsoleFrontClient>("Demo", MySession.class);

server.addApplication(application);

How to define a session class for console back clients

public class MySession extends Session<ConsoleBackClient> {
  public void initailize() {
    getRefClient().writeLineToConsole("Welcome!");
  }
}

How to create a console front client

var consoleFrontClient = new ConsoleFrontClient("200.200.200.200", 50000, "Demo");
