Nolix containers

Packages
ch.nolix.core.container
Usability Provides containers with various search methods.

How to create a read container for an array

String[] namesArray ...
ReadContainer<String> names = new ReadContainer<>(namesArray);
String longestName = names.getRefMax(n -> n.length());

How to get a subset of a container

List<Person> persons ...
List<Person> seniors = persons.getRefSelected(p -> p.getAge() >= 65);

How to get a value from the elements of a container

List<Person> persons ...
List<Person> ages = persons.to(p -> p.getAge());

How to get an average from the elements of a container

List<Person> persons ...
double averageAge = persons.getAverageByInt(p -> p.getAge());
