Nolix logger

Description

 Can log messages, values, warnings and errors.

Packages

 ch.nolix.primitive.logger

How to log a value

int amount ...
Logger.logValue("amount", amount);

How to log a warning

Logger.logWarning("There are many bananas.");

How to log an error

Logger.logError("There are too many bananas.");

How to disable the Logger at all

Logger.disable();

The logger will not log anything when it is disabled. The logger is enabled per default.


About the logger

The Nolix logger is a simple logger that is optimized for performance. When a log entry is created, it is added to a queue and the current thread can go on. The queue is processed in background, what means that the log entries are given to the log handlers.


What a log handler is

A log handler gets all log entries from the logger. A logger can filter the log entries and display them or write them to a file. The logger contains one log handler per default, which prints out the log entries to the standard console.
