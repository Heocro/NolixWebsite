Nolix Logger

Description

 Provides a Logger that can log messages, values, warnings and errors.

Packages

 ch.nolix.common.Logger


How to log a value

int amount;
...
Logger.logValue("amount", amount);

How to log a warning

Logger.logWarning("There are many bananas.");

How to log an error

Logger.logError("There are too many bananas.");

How to enable the Logger at all

Logger.enable();

The Logger will not log anything when it is disabled. The Logger is disabled per default.


About the Logger

The Nolix Logger is a simple Logger that is optimized for performance. When a log entry is created, it is added to a queue and the current thread can go on. The queue is processed in background, where the log entries are given to the log handlers.


What a LogHandler is

A LogHandler gets all log entries from the Logger. A Logger can filter the log entries and display them or write them to a file. At start, the Logger contains one default log handler, that prints out the log entries to the standard console.
